O objetivo principal desse projeto é te manter informado de uma forma simples e rápida com dados livres para que você possa ter melhor entendimento do processo de expansão da epidemia do COVID-19 no Brasil e no mundo.

Todas as informações desse aplicativo são retiradas de API's de terceiros que usam dados reportados por governos de todo o mundo para alimentar suas bases de dados.

A seguir são citadas as fontes de dados:

  • Dados gerais do Brasil e do mundo: ABOUT-CORONA.NET, com fonte de dados da OMS, Johns Hopkins CSSE e governos, departamentos e instituições de todo o mundo.

  • Dados dos estados e municípios brasileiros: Brasil.IO, com fonte de dados das Secretarias de Saúde das Unidades Federativas, e dados tratados por Álvaro Justen e colaboradores/Brasil.IO, sob a licença Creative Commons Attribution ShareAlike.

Caso seja de seu interesse o código fonte desse projeto está disponivel no GitHub.

Observação: pode haver pequenas discrepâncias de dados entre os dados gerais do Brasil e os dados estaduais e municipais quando somados e comparados, visto que, as fontes são de diferentes origens como citado acima.
