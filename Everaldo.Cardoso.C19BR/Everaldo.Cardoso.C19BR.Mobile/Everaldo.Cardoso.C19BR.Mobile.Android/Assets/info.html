Fontes de dados:

  • Dados gerais do Brasil e do mundo: ABOUT-CORONA.NET, com fonte de dados da OMS, Johns Hopkins CSSE e governos, departamentos e instituições de todo o mundo.

  • Dados dos estados e municípios brasileiros: Brasil.IO, com fonte de dados das Secretarias de Saúde das Unidades Federativas, e dados tratados por colaboradores do Brasil.IO, sob a licença Creative Commons Attribution ShareAlike.

Observações:

  • Pode haver pequenas discrepâncias de dados entre os dados gerais do Brasil e os dados estaduais e municipais quando somados e comparados, visto que, as fontes são de diferentes origens como citado acima.

  • A aplicação não mantém informações no aparelho, todos os dados são obtidos em modo on-line sendo obrigátorio acesso a internet.
